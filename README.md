# BitTorrent

## 项目概述

本项目为教学性质的 BitTorrent P2P 文件下载与传输工具的实现，基于 Kademlia DHT。

## 项目结构

本项目分为 DHT 部分和后端部分，未来可能会实现极简的前端。

本项目的后端逻辑基于 Rust 实现，DHT 基于 Go 实现，分别编译为两个分离的可执行文件：`backend` 和 `dht-sidecar`，二者通过 gRPC 在本机直接通信，实现了 DHT 与后端的解耦。

BitTorrent 的结点发现、组网、通信、退出等操作均由 DHT 完成。目前我们暂未支持 Tracker 定位，这可能是未来将会实现的功能。目前 BitTorrent 的各个结点需要处于同一局域网内，但本项目预留了对公网 IP 的支持接口。

后端主要包含 `.torrent` 文件解析、Peer 连接与通信、事件循环、分片下载逻辑（Tit-for-Tat 算法）、本地保存等功能。目前通过 CLI 启动。
